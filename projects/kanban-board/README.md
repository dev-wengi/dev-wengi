# 📋 Kanban Board

Build a simple task management board (like Trello).

---

## 🎯 Goal

Learn complex state management and UI interactions.

---

## 🧩 Requirements

- 3 columns:
  - Todo
  - Doing
  - Done
- Add tasks
- Move tasks between columns

---

## 🎯 Interaction

- Drag and drop (preferred)
OR
- Buttons to move tasks

---

## 🎨 UI Expectations

- Clear column separation
- Tasks should look like cards

---

## 🚫 Constraints

- No full UI frameworks

---

## ⭐ Bonus Challenges

- Save board state (localStorage)
- Add task editing
- Add multiple boards

---

## 📦 Submission

- PR with working interactions
- Demo or GIF preferred
