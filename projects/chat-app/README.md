# 💬 Chat App

Build a simple chat interface that simulates real-time communication using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

- Send messages instantly
- Simulated real-time replies
- Scrollable chat interface
- Chat bubble UI

---

## 🛠️ Getting Started

git clone <repo-url>  
cd chat-app  
open index.html  

No setup required.

---

## 📁 Structure

index.html  
styles.css  
script.js
