# ❓ Quiz App

Build an interactive quiz application.

---

## 🎯 Goal

Practice conditional rendering and managing application flow.

---

## 🧩 Requirements

- Display one question at a time
- Multiple choice answers
- Track score
- Show final results

---

## 🔁 Flow

1. Start quiz
2. Answer questions
3. Show score
4. Restart option

---

## 🎨 UI Expectations

- Clear question display
- Clickable answer options
- Feedback for selected answers (optional)

---

## 🚫 Constraints

- No external quiz libraries
- Questions can be hardcoded

---

## ⭐ Bonus Challenges

- Add a timer per question
- Fetch questions from an API
- Add animations between questions

---

## 📦 Submission

- Open a Pull Request
- Show your quiz in action (GIF/screenshot)
