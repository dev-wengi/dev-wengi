# 🌦️ Weather App

Build a weather app that fetches real-time data from an API.

---

## 🎯 Goal

Learn how to work with APIs and async data.

---

## 🧩 Requirements

- Input field for city name
- Fetch weather data from an API
- Display:
  - Temperature
  - Weather condition
  - City name

---

## 🌐 API

- Use any public weather API (e.g., OpenWeather)

---

## ⚠️ Error Handling

- Handle invalid city names
- Handle failed API requests

---

## 🎨 UI Expectations

- Clear and readable layout
- Loading state while fetching data

---

## 🚫 Constraints

- No heavy frameworks for data fetching

---

## ⭐ Bonus Challenges

- Show weather icons
- Add 5-day forecast
- Detect user location

---

## 📦 Submission

- PR with working API integration
- Include API setup instructions
