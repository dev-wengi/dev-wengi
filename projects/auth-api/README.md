# 🔐 Authentication API

Build a basic authentication system.

---

## 🎯 Goal

Learn how authentication and security work in backend systems.

---

## 🧩 Requirements

- Register user
- Login user

---

## 🔁 Endpoints (Suggested)

- `POST /register`
- `POST /login`
- `GET /profile` (protected)

---

## 🔒 Features

- Password hashing
- Token-based authentication (JWT or similar)

---

## 🗂️ User Model

- id
- email
- password (hashed)

---

## ⚠️ Error Handling

- Invalid credentials
- Duplicate users
- Unauthorized access

---

## 🚫 Constraints

- Do not store plain-text passwords

---

## ⭐ Bonus Challenges

- Refresh tokens
- Logout functionality
- Role-based access

---

## 📦 Submission

- PR with working auth flow
- Explain how to test protected routes
