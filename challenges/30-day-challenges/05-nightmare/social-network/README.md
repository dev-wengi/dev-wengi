# Social Network

Category: Nightmare

## Project Overview

Build a Social Network that supports user profiles, posts, and social interactions with a polished social experience. The application should replicate core social media functionality in a simplified but engaging interface.

## Core Features

- Create and view user profiles with bio, avatar, and personal information
- Post updates or content with text, images, or links
- Like, comment, or interact with posts from other users
- Follow or connect with other users to build a social graph
- Display a responsive, social feed interface with infinite scroll
- Optional: Direct messaging between users

## Technical Recommendations

- Frontend: React, Vue, or Next.js with a social feed and profile layout
- Backend: Node.js with Express or Next.js API routes for social features
- Database: PostgreSQL or similar for storing users, posts, likes, and follows
- Real-time: Socket.IO or WebSockets for live feed updates and notifications
- State Management: React Context, Zustand, or Vuex for complex social state
- Styling: Tailwind CSS or CSS Modules for clean, social-inspired layouts
- Media: Implement image upload and storage for post media

## Success Criteria

- Users can share and interact with content through posts and reactions
- The social experience feels cohesive with clear user connections
- The app demonstrates a complete mini social platform
- User profiles and feeds are informative and engaging
- The interface is responsive and accessible
