# Weather Dashboard

Category: Easy

## Project Overview

Build a Weather Dashboard that lets users search for a location and display current weather conditions with a clean, informative layout. The application should provide real-time weather data in an easy-to-read format.

## Core Features

- Search for a city or location
- Display current temperature, conditions, and weather details
- Show additional weather data such as humidity, wind speed, and pressure
- Optional: Display a 5-day forecast
- Responsive design for different screen sizes
- Graceful handling of invalid locations or network issues

## Technical Recommendations

- Frontend: React, Vue, or vanilla JavaScript
- Weather API: OpenWeatherMap API, WeatherAPI, or similar service
- HTTP Client: fetch API or Axios for API requests
- State Management: React hooks, Vue Composition API, or vanilla JS
- Styling: Tailwind CSS or CSS Modules for clean, card-based layouts
- Icons: Weather icon library or SVG icons for conditions
- Error Handling: Implement loading states and error messages

## Success Criteria

- Users can quickly get weather information by location
- The app presents weather data clearly and attractively
- The interface adapts well on mobile and desktop
- Error states are handled gracefully
- The application provides a smooth user experience
