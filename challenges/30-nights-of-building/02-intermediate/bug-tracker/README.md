# Bug Tracker

Category: Intermediate

## End of 30-Day Goal

This Bug Tracker should let users report issues, update statuses, and track bug progress in a simple issue management board.

## Expected Features

- Log new bug reports with details.
- Update bug status and priority.
- View a list of issues with filters.
- Responsive issue tracker layout.

## Success Criteria

- Users can manage bugs clearly.
- The issue workflow is easy to follow.
- The app feels complete for basic bug tracking.
