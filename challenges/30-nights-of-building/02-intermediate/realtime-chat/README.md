# Realtime Chat

Category: Intermediate

## End of 30-Day Goal

This Realtime Chat should support live message exchange between users with an instantly updating conversation view.

## Expected Features

- Send and receive messages in real time.
- Display conversation history.
- Show participant presence or typing state if possible.
- Responsive chat interface.
- Reliable message flow with clean input handling.

## Success Criteria

- Users can chat live without page refresh.
- Messages appear clearly and in order.
- The interface is comfortable for conversation.
