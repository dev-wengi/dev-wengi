# Kanban Board

Category: Intermediate

## End of 30-Day Goal

By the end of thirty days, this Kanban Board should enable users to organize tasks into columns, move cards between stages, and manage workflows visually.

## Expected Features

- Create task cards with titles and descriptions.
- Drag and drop cards between columns.
- Add, edit, and delete cards or columns.
- Persist board state locally.
- A responsive, flexible layout.

## Success Criteria

- Users can manage workflow stages intuitively.
- The board supports dynamic task movement.
- The app feels stable and polished for task organization.
