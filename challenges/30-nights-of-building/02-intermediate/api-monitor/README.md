# API Monitor

Category: Intermediate

## End of 30-Day Goal

By the end of thirty days, this API Monitor should let users check endpoint status and review API health in a monitoring dashboard.

## Expected Features

- Monitor API endpoints for availability.
- Show response status and uptime summaries.
- Display recent check results.
- Responsive layout for monitoring at a glance.

## Success Criteria

- Users can track API status easily.
- Monitoring data is presented clearly.
- The dashboard supports simple reliability checks.
