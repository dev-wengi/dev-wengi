# Search Engine

Category: Hard

## End of 30-Day Goal

This Search Engine should index content and handle queries, returning relevant ranked results for search terms.

## Expected Features

- Index documents or data for search.
- Accept query input and search indexed content.
- Return matched results ranked by relevance.
- Display search snippets or highlights.
- A helpful search interface.

## Success Criteria

- Users can search and discover relevant content.
- The search engine returns meaningful results.
- The implementation shows a functioning search pipeline.
