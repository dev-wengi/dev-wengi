# Distributed Job Runner

Category: Hard

## End of 30-Day Goal

This Distributed Job Runner should break work into jobs, distribute them across workers, and track execution progress.

## Expected Features

- Submit jobs or tasks to the system.
- Distribute work to available workers.
- Monitor job status and completion.
- Handle retries or failures gracefully.
- Show worker and job metrics.

## Success Criteria

- Jobs are executed across multiple workers.
- Progress is visible and manageable.
- The system illustrates distributed job coordination.
