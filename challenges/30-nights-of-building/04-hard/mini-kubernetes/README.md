# Mini Kubernetes

Category: Hard

## End of 30-Day Goal

This Mini Kubernetes project should simulate basic container orchestration, scheduling workloads and managing service connectivity.

## Expected Features

- Schedule containerized workloads or jobs.
- Manage simple service discovery between components.
- Monitor pod or container state.
- Support deployment and lifecycle operations.
- Present the orchestration model clearly.

## Success Criteria

- The system demonstrates core orchestration behavior.
- Workloads are scheduled and managed consistently.
- The implementation communicates the Kubernetes concept effectively.
