# Message Queue

Category: Hard

## End of 30-Day Goal

This Message Queue should provide reliable messaging between producers and consumers with ordering and delivery behavior.

## Expected Features

- Enqueue messages from producers.
- Dequeue messages for consumers.
- Support message acknowledgement or retry behavior.
- Track queue status and processing metrics.
- A clear interface for queue operations.

## Success Criteria

- Messages flow reliably from producers to consumers.
- The queue supports basic delivery guarantees.
- The implementation shows a practical message queuing system.
