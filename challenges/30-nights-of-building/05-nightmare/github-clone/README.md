# GitHub Clone

Category: Nightmare

## End of 30-Day Goal

This GitHub Clone should simulate a repository browsing experience with profiles, repo listings, and basic actions.

## Expected Features

- View repositories and repository details.
- Browse user profiles and activity.
- Display issues, pull requests, or repo metadata.
- A clean code-hosting interface.
- Responsive navigation for repo exploration.

## Success Criteria

- Users can explore projects and profiles easily.
- The clone feels familiar and usable.
- The app demonstrates the core repository browsing experience.
