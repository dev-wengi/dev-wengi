# Mini Cloud Provider

Category: Nightmare

## End of 30-Day Goal

This Mini Cloud Provider should let users manage resources, deploy apps, and simulate core cloud infrastructure services.

## Expected Features

- Provision virtual resources or services.
- Deploy applications or workloads.
- Display resource status and usage.
- Provide a simple cloud management dashboard.
- A clean interface for infrastructure operations.

## Success Criteria

- Users can create and manage cloud-like resources.
- The system communicates cloud infrastructure concepts clearly.
- The app feels like a working mini-cloud control plane.
