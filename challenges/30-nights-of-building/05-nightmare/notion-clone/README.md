# Notion Clone

Category: Nightmare

## End of 30-Day Goal

This Notion Clone should let users create and organize notes, pages, and databases like a productivity workspace.

## Expected Features

- Create pages or documents with rich content.
- Organize pages in a workspace structure.
- Support editing and formatting content.
- Search or navigate between pages.
- A polished productivity interface.

## Success Criteria

- Users can build and navigate a workspace.
- The experience feels organized and flexible.
- The app resembles a simplified Notion-style editor.
