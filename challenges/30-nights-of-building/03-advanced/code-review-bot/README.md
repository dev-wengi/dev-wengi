# Code Review Bot

Category: Advanced

## End of 30-Day Goal

This Code Review Bot should evaluate code snippets, identify issues, and provide constructive feedback for improvement.

## Expected Features

- Accept code input from users.
- Analyze code for style, logic, and best practices.
- Offer clear review comments and suggestions.
- Highlight potential bugs or maintainability concerns.
- A user-friendly review interface.

## Success Criteria

- Users receive helpful code review feedback.
- The tool explains issues clearly.
- The final app feels like a trustworthy code review assistant.
