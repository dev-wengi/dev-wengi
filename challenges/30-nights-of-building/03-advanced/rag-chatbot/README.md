# RAG Chatbot

Category: Advanced

## End of 30-Day Goal

By the end of thirty days, this RAG Chatbot should answer questions using a combination of retrieved documents and generative responses.

## Expected Features

- Accept user queries in a chat interface.
- Retrieve relevant information from a document collection.
- Combine retrieval results with generated answers.
- Show citations or references from source documents.
- A conversational interface that feels responsive.

## Success Criteria

- Users receive accurate, context-aware responses.
- The bot leverages documents effectively.
- The experience feels like a helpful knowledge assistant.
