# Meeting Summarizer

Category: Advanced

## End of 30-Day Goal

This Meeting Summarizer should let users submit notes or transcripts and return concise summaries of the main points and action items.

## Expected Features

- Accept meeting notes, transcripts, or audio text.
- Generate a clear summary of key points.
- Identify action items or decisions.
- Present the summary in a readable format.
- A polished interface for quick meeting review.

## Success Criteria

- Users get a useful summary of meetings.
- The output is structured and easy to follow.
- The tool feels like a practical productivity helper.
