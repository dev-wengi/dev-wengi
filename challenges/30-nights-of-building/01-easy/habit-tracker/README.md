# Habit Tracker

Category: Easy

## End of 30-Day Goal

By the end of the thirty-day challenge, this Habit Tracker should be a polished daily habit management app where users can add habits, mark them as complete, and view progress over time.

## Expected Features

- Add, edit, and remove habits.
- Mark habits as completed for the current day.
- Display streaks or completion indicators.
- Persist habit data locally so progress remains after refresh.
- A clean and responsive interface that works on mobile and desktop.

## Success Criteria

- Users can quickly track their daily routines.
- The app preserves state in the browser.
- The UI is intuitive enough for regular daily usage.
