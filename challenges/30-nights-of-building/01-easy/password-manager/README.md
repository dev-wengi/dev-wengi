# Password Manager

Category: Easy

## End of 30-Day Goal

By the end of the challenge, this Password Manager should store login credentials securely and allow users to manage passwords in a simple interface.

## Expected Features

- Add new password entries with site name, username, and password.
- Hide passwords by default with a reveal option.
- Edit and remove stored credentials.
- Preserve data locally between sessions.
- A clean and reassuring UI.

## Success Criteria

- Users can manage credentials easily.
- Password entries are presented securely and clearly.
- The UI supports a strong yet approachable design.
