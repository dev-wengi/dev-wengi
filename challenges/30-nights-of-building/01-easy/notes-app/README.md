# Notes App

Category: Easy

## End of 30-Day Goal

This Notes App should allow users to create, edit, delete, and persist notes in a simple note-taking interface.

## Expected Features

- Add new notes with titles and content.
- Edit and remove existing notes.
- Persist notes locally so they remain after refresh.
- Search or filter notes if appropriate.
- Clean and responsive design.

## Success Criteria

- Users can manage notes without confusion.
- Note data is saved reliably.
- The app feels complete and ready for regular use.
