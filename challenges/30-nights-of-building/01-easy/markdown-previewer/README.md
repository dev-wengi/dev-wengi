# Markdown Previewer

Category: Easy

## End of 30-Day Goal

By the end of thirty days, the Markdown Previewer should let users write Markdown content and instantly see a rendered preview side by side.

## Expected Features

- Text editor input for Markdown.
- Live preview rendering the formatted Markdown.
- Support for headings, links, lists, code blocks, and emphasis.
- Responsive layout that works in narrow and wide views.
- Clean separation between input and preview panes.

## Success Criteria

- Users can write Markdown and see the output immediately.
- The preview is accurate and well-formatted.
- The app is easy to use and visually polished.
