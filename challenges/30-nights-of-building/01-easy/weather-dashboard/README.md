# Weather Dashboard

Category: Easy

## End of 30-Day Goal

This Weather Dashboard should let users search for a location and display current weather conditions with a clean, informative layout.

## Expected Features

- Search for a city or location.
- Display current temperature, conditions, and weather details.
- Show additional weather data such as humidity, wind, and forecast highlights.
- Responsive design for different screen sizes.
- Graceful handling of invalid locations or network issues.

## Success Criteria

- Users can quickly get weather information by location.
- The app presents weather data clearly and attractively.
- The interface adapts well on mobile and desktop.
