# QR Generator

Category: Easy

## End of 30-Day Goal

At the end of thirty days, this QR Generator should let users create QR codes from text or URLs and present the result clearly.

## Expected Features

- Input text or a URL to generate a QR code.
- Display the QR image immediately.
- Allow downloading or copying the QR code if available.
- Provide error feedback for invalid input.
- Responsive design for mobile and desktop.

## Success Criteria

- Users can generate a valid QR code instantly.
- The output is easy to use and visually clear.
- The interface is polished and functional.
