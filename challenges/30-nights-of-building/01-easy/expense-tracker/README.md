# Expense Tracker

Category: Easy

## End of 30-Day Goal

This Expense Tracker should enable users to log expenses, categorize spending, and review totals in a simple financial dashboard.

## Expected Features

- Add new expense entries with amount and category.
- View a list of recorded expenses.
- Show totals or category summaries.
- Edit or delete existing expense items.
- Responsive design for desktop and mobile screens.

## Success Criteria

- Users can track spending in an intuitive way.
- The app displays expense data clearly.
- The UI is polished and the main workflow works smoothly.
